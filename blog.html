Question-1: What are the different ways to select an element in the DOM?

DOM works with different types of elements.Example:-
1. getElementById: Selects a single element using its unique id. Best for targeting specific elements.
2. getElementsByClassName: Selects all elements that share the same class name. Returns a live collection of elements.
3. getElementsByTagName: Selects all elements with the given tag name (returns an HTMLCollection).
4. getElementsByNameAttribute: Selects elements based on the name attribute, mainly used for form elements like input fields.
5. quarySelectorAll:Selects all elements that match a CSS selector and returns them as a static list.
6.quarySelector: Selects the first element that matches a CSS selector, such as a class, ID, or tag.
